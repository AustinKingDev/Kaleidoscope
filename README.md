# Kaleidoscope
Uses an image to create a kaleidoscope effect


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build

```

### Compiles and Watches JS Files

```
npm run watch

```
